# Image Converter

Библиотека для конвертации изображений между различными форматами с поддержкой алгоритмов обработки изображений.

## Описание

Библиотека для конвертации изображений между различными форматами с поддержкой различных алгоритмов обработки изображений и фильтров. Позволяет конвертировать изображения с сохранением качества и применением различных фильтров.

## Функциональность

- Конвертация между различными форматами изображений
- Применение фильтров обработки изображений
- Работа с бинарными данными
- Модульная архитектура

## Технологии

- **C++17** - основной язык программирования
- **Работа с бинарными данными** - чтение и запись изображений
- **Алгоритмы обработки изображений** - различные фильтры и преобразования
- **CMake** - система сборки

## Структура проекта

```
image-converter/
```

## Требования

- Компилятор с поддержкой C++17
- CMake 3.10+

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Использование

### Базовое использование

```cpp
#include "image.h"
#include "converter.h"

// Загрузка изображения
Image img = Image::load("input.png");

// Конвертация в другой формат
img.save("output.jpg");

// Применение фильтра
img.applyFilter(FilterType::Grayscale);
img.save("output_grayscale.jpg");
```

### Поддерживаемые форматы

- PNG
- JPEG
- BMP
- (и другие, в зависимости от реализации)

### Доступные фильтры

- Grayscale (оттенки серого)
- Blur (размытие)
- Sharpen (резкость)
- (и другие, в зависимости от реализации)

## Особенности реализации

- **Модульная архитектура**: легко добавлять новые форматы и фильтры
- **Эффективная обработка**: оптимизированные алгоритмы
- **Сохранение качества**: минимизация потерь при конвертации
